# CCAP — Colegio de Contadores y Auditores de Pichincha

Sitio web profesional para el **Colegio de Contadores y Auditores de Pichincha (CCAP)**, el gremio contable más representativo del Ecuador.

## 🚀 Stack Tecnológico

- **React 18** — UI framework
- **Vite 4** — Build tool y dev server
- **Tailwind CSS 3** — Utility-first CSS
- **Framer Motion** — Animaciones y micro-interacciones
- **Lucide React** — Iconografía

## 📦 Setup

```bash
npm install
npm run dev       # http://localhost:5173
npm run build     # /dist
```

## 🗂 Estructura

```
src/
├── components/    # Navbar, Hero, Services, About, Events, Membership, Contact, Footer
│   └── ui/        # SocialIcons
├── data/          # data.json
├── hooks/         # useScrollAnimation
└── index.css      # Design system
```
